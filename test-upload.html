Android Studio Integration

Upload API Endpoint:

POST https://communitybusinessservices.ca/api/android/upload Content-Type: multipart/form-data Body: - image: (File) The image to upload - deviceModel: (String) Optional device information - androidVersion: (String) Optional Android version

Retrofit Example (Kotlin):

interface ImageUploadApi { @Multipart @POST("/api/android/upload") suspend fun uploadImage( @Part image: MultipartBody.Part, @Part("deviceModel") deviceModel: RequestBody? = null, @Part("androidVersion") androidVersion: RequestBody? = null ): Response<UploadResponse> } // Usage: val file = File("/path/to/image.jpg") val requestFile = file.asRequestBody("image/*".toMediaType()) val imagePart = MultipartBody.Part.createFormData("image", file.name, requestFile) val response = api.uploadImage(imagePart)
